# 月次請求書 半自動作成

eBay外注3名（本間・清水・佐々木）の月次請求書を「収支表（KeiS）2026」「仕入管理表」「【佐々木さん】実績管理表」をまたいで半自動で作成する。社長は最終チェックとPDF出力のみ。

引数 `$ARGUMENTS` はYYYYMM形式（例：`202603`）。省略時は前月。

---

## 実行手順

### Step 1: ドライラン実行

`commerce/ebay/staff-ops/invoice-automation/` で以下を実行：

```bash
python run.py --month $ARGUMENTS
```

引数が空なら `python run.py` （前月分）を実行。

ドライラン結果は標準出力に表示される。書込予定の値（本間の単価・清水の仕入管理表AH値・佐々木の推定インセンティブ）を社長が確認できる形で出る。

### Step 2: 結果を社長に報告

ドライラン結果から以下を抜き出して、整理した形で社長に提示する：

- **本間**：書込先セル・単価（通常10,000円）・件名・日付
- **清水**：仕入管理表のどの行から単価を取得したか・取得した値・書込先
- **佐々木**：
  - 当月SKU=S抽出件数
  - 前月までの累計U・当月推定U・推定累計
  - 5万切捨後の累計万円・当月対象万円・推定インセンティブ円
  - 備考予定文面
  - 実績管理表の書込予定行範囲（既存データの有無）

警告（`⚠️`）が出ていたら必ず社長に報告し、判断を仰ぐ。

社長に「上記内容で実行してよいか」を確認する。

### Step 3: 社長OK後、実書込

社長OKを受けたら：

```bash
python run.py --month $ARGUMENTS --apply
```

書込時は以下の順で実行される：
1. 実績管理表に明細N行追記
2. 集計行作成（=COUNTA, =SUM, =S/R, =S*0.95）
3. G3/I3末尾に新月参照を追記
4. I3確定値で再計算してインセンティブ・備考を確定
5. 各請求書ブロック（AC:AP等）にコピー＋件名・日付・単価・備考を上書き

### Step 4: 完了報告

実行後、3スプレッドシートをstartコマンドで開く（社長が画面で確認できるように）：

```bash
start "" "https://docs.google.com/spreadsheets/d/1t_GHxn2PQY2F78Xgx9B0o602Wl0CeJSeBlA1Wo1ngME/edit"
start "" "https://docs.google.com/spreadsheets/d/1cKiRQIUHb2LM0p8yrxkQ4voorEkjGgprf4ox8WOrrKY/edit"
```

完了報告には以下を含める：
- 確定したインセンティブ金額・備考文面（佐々木）
- 書込先セル一覧
- ログJSONファイル名（`commerce/ebay/staff-ops/invoice-automation/logs/`配下）
- 社長への次アクション：「単価微調整・PDF出力をお願いします」

## エラー時の対応

- **シート名エラー**：当月の収支表シートが未作成。社長に確認。
- **権限エラー**：`reffort-claude@reffort-sheets.iam.gserviceaccount.com` が編集者付与されているか確認。
- **既存データ警告**：実績管理表の新月集計行範囲に既存データがあると停止する。手動確認のうえ、ロールバック判断を社長に仰ぐ。

ロールバックが必要な場合は **Google Sheetsの版履歴** から復元可能。
