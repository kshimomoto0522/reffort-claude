# memory/archive/ 配下のルール

- ここに置かれたファイルはClaudeが通常のセッションで参照しません
- 完了済みプロジェクト・部門CLAUDE.mdや退避ファイルに情報保全済みのmemoryの保存場所
- 削除はしない。情報資産として保全
- 必要な時は社長が明示的に「archive/○○.md を読んで」と指示

## archive 移動の基準

- 完了した、または進行が停止したプロジェクト系memory
- 部門CLAUDE.md／退避ファイル／decisions_log.md に内容が移管されてmemoryでの保持が冗長なもの
- 過去の失敗教訓など、現在のアクティブ参照がないもの

## 復活させる場合

- `mv archive/{ファイル}.md ../` で戻す
- MEMORY.md index に再追加
