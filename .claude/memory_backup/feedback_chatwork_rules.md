---
name: Chatwork返信ルール
description: AIがChatworkグループチャットで返信・反応するメッセージの判断基準
type: feedback
---

AIへのメンション または AIへの返信（スレッド返信）があった場合のみ反応する。
それ以外（社長・スタッフ間のやりとり、画像確認依頼など）はスルー。

**Why:** 関係ないやりとりに首を突っ込まないようにするため。社長が「画像レビューは無視でOK」と言っていたが、それはAI宛てでないから。
**How to apply:** チャットワーク確認時、AI（+REFFORT AI+）へのメンション・返信があるメッセージだけを「対応が必要なもの」として報告する。
