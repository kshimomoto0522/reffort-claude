事業のAI活用実践（特にeBay広告最適化のような取り組み）は、発見・テスト・結果のすべてをコンテンツ化前提で記録すること。

**Why:** 社長が長年eBayをやっていても広告費の実態を把握できていなかったように、多くのセラーが同じ問題を抱えている。この実体験はCampers有料コンテンツ・コンサル事業・新Bay系サービス・X/Note発信に転用できる。

**How to apply:**
- 各セッションで「これはコンテンツになる」と判断した発見・データ・テスト結果は、journey-log.mdに【コンテンツ候補】タグ付きで記録
- 発信先は個人X / BayChat公式X / Note / ショートムービーを想定（最終判断は社長）
- 口調は後から調整可能なので、まず「事実・発見・数字・ストーリー」を中立的に記録
- 提案時は「このネタはXで反応取れそう」等のコメントも添える
- journey-log.mdとは別に、SNS投稿ネタのストックも意識する
