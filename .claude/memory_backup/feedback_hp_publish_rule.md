本番公開されているHP（reffort.co.jp）への変更は、必ず以下の手順を守ること。

1. CSSや変更内容を準備する
2. スクリーンショット（プレビュー）を社長に見せる
3. **「公開していいですか？」と明示的に確認する**
4. OKをもらってから初めて「公開」ボタンを押す

**Why:** 確認なしに本番サイトを変更してしまい、一般公開されているサイトに無断で変更を加えた。社長から強く指摘を受けた（2026-03-31）。

**How to apply:** WordPressカスタマイザーの「公開」ボタンは、社長の明示的なOKなしには絶対に押さない。プレビューはカスタマイザーのプレビュー画面のスクリーンショットを使う。
