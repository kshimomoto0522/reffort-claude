# 用語集

| 用語 | 意味 |
|------|------|
| BayChat | Reffort運営のeBayセラー向け顧客対応SaaS |
| BayPack | Reffort運営のeBay返品代行SaaS（アメリカ倉庫あり） |
| Campers | Reffort運営のeBay輸出スクール（約70名の生徒） |
| Cowatech | BayChat・BayPackの開発を担当するベトナムの開発会社 |
| AI Reply | BayChatに実装中のAI返信自動生成機能。数か月以内のリリースが目標 |
| eBay API | eBayが提供する公式データ連携インターフェース |
| セラー | eBayで商品を販売している出品者 |
| バイヤー | eBayで商品を購入する買い手 |
| CS | カスタマーサポートの略 |
| 一括納品 | BayPackの非公開サービス。日本在庫をまとめてBayPack倉庫へ納品するモデル |
| コンバージョン率 | 出品した商品が実際に売れた割合。パフォーマンスの核心指標 |
| Cowatech仕様書 | 新機能をCowatechに発注する際に渡す日本語の開発指示書 |
