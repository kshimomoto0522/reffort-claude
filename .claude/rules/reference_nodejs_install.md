## Node.jsインストール時のバージョン選択

**→ 必ず「LTS版」を選ぶ**

- LTS = Long Term Support（長期サポート版）
- 安定していて業務用途はこれ一択
- 公式サイト（https://nodejs.org/）の緑の「Node.js®を入手」ボタンを押すと自動的にLTS版がダウンロードされる
- 「最新版」は新機能が入っている代わりに不安定なことがあるため使わない

### よく聞かれるパターン

**「どのバージョンを選ぶ？」**
→ 左側の「LTS版」（例：v24.14.1 LTS版）を選んでください。

**「ダウンロードリンクがたくさんあってどれを押す？」**
→ Windowsの場合は「Windows 64-bit Installer」（.msiファイル）を選んでください。
　 それ以外（Binary、ARM、macOS、Linux等）は不要です。
