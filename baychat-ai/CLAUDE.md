## サービス概要

| 項目 | 内容 |
|------|------|
| サービス名 | BayChat |
| 種別 | eBayセラー向け顧客対応・CSツール（SaaS） |
| 形態 | Webブラウザアプリ（PC・スマホ対応） |
| 認定 | eBay Compatible Application（公認） |
| 開発 | Cowatech（ベトナム） |
| 特徴 | シンプルイズザベスト。CSに特化。統合ツール化はしない方針 |

## 主要機能一覧

| 機能 | 内容 |
|------|------|
| 翻訳 | バイヤーメッセージを自動翻訳。日本語→各言語に翻訳して送信。DeepL・Google翻訳から選択可能 |
| テンプレート | よく使う返信文を登録・呼び出し。eBayアカウントごとに設定可能 |
| クーポン | eBayで作成したクーポンコードをチャット画面から表示・送信 |
| リピーター管理 | リピーターマーク表示・購入履歴一覧・過去のやり取り（Chat History）確認 |
| タスク管理 | バイヤー/オーダーごとにチームでタスク共有。担当者・期限設定可能 |
| 商品編集 | 価格・数量・Best Offer・出品取消をチャット画面から直接編集 |
| 自動メッセージ | 売れた時・発送時・配達時・フィードバック時・営業時間外など（詳細は下記） |
| AI Reply | **開発中**（テスト環境のみ。本番リリース準備中） |

## 自動メッセージ機能（詳細）

### 送信タイミング一覧

| タイミング | 内容 |
|----------|------|
| 商品が売れた時 | すぐ〜24時間後まで指定可能。リピーター専用メッセージも設定可能 |
| オファーが成立/拒否した時 | オファー結果に応じて自動送信 |
| 商品を発送した時 | Mark as shippedが対象 |
| 商品が配達された時 | eBay配送状況を反映 |
| 配達完了のN日後 | 1〜90日後まで設定可能 |
| フィードバックをもらった時 | ポジティブ・ニュートラル・ネガティブ別々に設定可能 |
| 営業時間外 | 曜日ごとに設定。定休日はその曜日の24時間。タイムゾーン設定対応 |

### 共通設定・その他
- トラブル（未着・リターン・ケース・Dispute）が発生したオーダーへの自動送信を停止できる
- 対象外バイヤーの個別設定が可能
- インサート機能：バイヤーID・氏名・オーダーID・商品名・追跡番号・配送先住所等を自動挿入

## ユーザー数・売上データ

### 用語定義
- **Active**: 無料期間中ユーザーを含む全登録ユーザー数
- **Inactive**: 無料終了後に有料契約しなかった、または解約済みユーザー数
- **Total Payment**: 月次売上（円）
- **No plan**: その月の無料ユーザー数（Activeの内訳）
- **Termination**: 解約者数

### トレンドサマリー（バグのある3月データを除く）

| 時期 | Active | 月次売上 | 状況 |
|------|--------|---------|------|
| 2024年初 | ~300 | ~¥600,000 | サービス立ち上げ期 |
| 2024年末 | ~455 | ~¥1,600,000 | 順調に成長 |
| 2025年ピーク（8月） | 583 | ¥2,204,500 | 過去最高 |
| 2025年末 | ~529 | ~¥2,094,800 | やや減少 |
| 2026年2月 | 527 | ¥1,878,600 | 減少傾向に注意 |

### プラン別内訳（2026年2月時点）
- STANDARD: 322名（最多）
- PREMIUM: 180名
- PREMIUM+: 12名（PREMIUM内でアカウント追加契約）
- 無料30日: 17名
- 無料60日: 8名

### 課題・懸念点
- Inactiveユーザーが増加し続けている（2024年1月：21名 → 2026年2月：643名）
- → 無料トライアル後の有料転換率の改善が必要
- Activeユーザーがピーク（583）から減少傾向（504〜527）
- 3月データはシステムバグあり（Monthlyに異常値が出る等）→ 信頼性に注意

## 競合分析

### 主要競合

#### ① Bee（Awiiin社）← 現時点で唯一の直接競合
- URL: https://bee-group.net/
- **強み**: すでにAI予測返信を実装済み。統合ツールとして進化中
- **弱み**: 高価格・UIがチープで使いづらい・認知度が低い・会員数は微々たるもの
- **マーケ**: メルマガ・Youtubeでこまめに発信しているが効果は限定的
- **注意点**: 統合ツール化を進めており、無視できない存在

#### ② ネギ（YouTuber・eBay Japan Awards 3年連続受賞）← 近日参入予定
- **概要**: eBay業界で影響力あるYoutuber。昨年（2024年）、類似ツールを**無料で提供する**と発言
- **狙い**: ツール登録者をコンサル等に誘導するビジネスモデル
- **懸念**: 無料＋影響力のある人物のため、認知・ユーザー獲得は速い可能性あり
- **対応策**: BayChatの質の高さ・AIの精度で差別化

#### ③ 未確認の第三者開発者（小耳に挟んだ情報）
- 詳細不明
- BayChatの認知がある中で参入するということはAI全面活用で一気に奪取を狙っている可能性大

#### ④ 3D Sellers（アメリカ）
- 日本人向けではないため現時点では競合として意識しない

### 競合に対するBayChatの戦略
- **シンプルイズザベスト**: 統合ツール化はしない。CSに特化し使いやすさを維持
- **AI Replyの質**: Beeより高品質なAI返信で差別化
- **認知・信頼**: eBay Japan Awards受賞・既存ユーザーの支持を活かす
- **UI/UX**: BeeのチープなUIに対してクリーンで使いやすいUIを維持
