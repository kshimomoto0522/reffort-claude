# 仕入れ用VPS（仮想PC）運用マニュアル

## 佐藤さん向け — 仕入れ時のIP・端末制限対策

---

## このVPSとは？

クラウド上にある「別のPC」です。自分のPCからリモート操作で使います。
- IPアドレスが自分のPCとは完全に別
- 端末情報（ブラウザ指紋）も完全に別
- 通販サイトから見ると「別の場所・別のPCからアクセスしている人」に見える

---

## 日常の使い方（仕入れ時）

### 接続する

1. PCのスタートメニューで「**リモート**」と検索
2. **「リモート デスクトップ接続」**を開く
3. 以下を入力して「接続」をクリック：
   - コンピューター: **（IPアドレスは下の「現在のサーバー情報」を参照）**
   - ユーザー名: `Administrator`
4. パスワードを入力 → VPSのWindows画面が開く
5. VPS上のブラウザ（EdgeまたはChrome）で仕入れサイトにアクセス

### 仕入れが終わったら

- リモートデスクトップの **×ボタン** を押して閉じるだけでOK
- VPSは裏で動き続けるが、問題なし（次回すぐ接続できる）

### 接続のコツ

- 「オプションの表示」→「全般」タブでユーザー名を入力し、「保存」しておくと次回から楽
- 「資格情報を保存する」にチェックを入れるとパスワード入力も省略できる

## 環境がダメになったら（IPがブロックされた等）

### サーバーを削除する

1. **ConoHa管理画面**にログイン → https://cp.conoha.jp/
2. 左メニュー「サーバー」をクリック
3. 該当サーバーのネームタグをクリック
4. **「サーバー削除」**（赤いゴミ箱ボタン）をクリック
5. 確認で「はい」→ 削除完了（課金も即停止）

### 新しいサーバーを作る

1. ConoHa管理画面で **「サーバー追加」**（左上の緑ボタン）をクリック
2. 以下の設定を選択：

| 項目 | 選択する値 |
|------|-----------|
| サービス | **Windows Server** |
| イメージタイプ | **Remote Desktop** → 2025 (x86_64) |
| 料金タイプ | **時間課金** |
| プラン | **1GB**（1,391円/月） |
| ライセンス RDS SAL | **追加申込数: 0個**（既に1個所持しているのでそのまま） |
| ライセンス Office SAL | **0個** |
| Administrator パスワード | 新しいパスワードを設定（必ずメモ） |
| オプション | すべてそのまま（自動バックアップ:無効、追加ストレージ:使用しない） |

3. **「追加」**をクリック → 数分でサーバーが起動
4. サーバーリストに新しいIPアドレスが表示される
5. そのIPアドレスでリモートデスクトップ接続

## よくある問題と対処法

### リモートデスクトップが繋がらない

1. ConoHa管理画面でサーバーのステータスが「起動中」か確認
2. 「起動中」なのに繋がらない → **「再起動」ボタン**を押して2〜3分待ってから再接続
3. それでもダメ → サーバーを削除して作り直す（5分で完了）

### パスワードがわからなくなった

- ConoHa管理画面からパスワードの確認はできない
- サーバーを削除して新しく作り直すのが一番早い

### 「シャットダウンイベントの追跡ツール」が表示された

- 再起動後に出るWindowsのメッセージ
- **「キャンセル」**を押して閉じればOK

---

## 注意事項

- VPS上では**仕入れに必要なサイトだけ**を使うこと
- VPSごとに**別のアカウント**で仕入れサイトにログインすること
- サーバーは**使わなくなったら必ず削除**すること（存在するだけで課金される）
- RDS SALライセンス（月1,386円）は**サーバーを削除しても課金が続く**。不要になったらライセンス画面で削除すること
