# 02. 出品 — 商品ページ作成・登録業務のAI化

> このフォルダは「商品をeBayに登録する」出品業務をAI化する事例を集める棚です。

---

## この業務段階で何ができるか（メンバー向け）

### 課題（メンバーが抱えがちな悩み）
- 出品作業に時間がかかる（タイトル考える・写真整える・説明文書く・カテゴリ決める）
- 同じような商品でも毎回手作業
- スタッフに任せるとミス（型番違い・サイズ違い・カテゴリ間違い）
- タイトルのSEO最適化が分からない
- 多言語対応がしんどい（日本人セラーは英語が課題）

### Claude Code でできること
- **タイトル自動生成**（売れ筋キーワード分析 → 最適タイトル提案）
- **商品説明文の自動生成**（型番・サイズ・状態・写真情報から）
- **カテゴリ自動判定**（商品名と特徴から最適カテゴリ）
- **多言語対応**（日本語入力 → 英語タイトル/説明文）
- **複数バリエーション一括登録**（サイズ/カラー違いをスプレッドシートから一括出品）
- **既存出品のSEO分析**（CTR低い商品のタイトル改善提案）

## メンバーが最初に試すべきこと

1. **過去30日の出品作業を棚卸し**（何件出品したか・1件あたり何分か）
2. **最も時間がかかる工程を特定**（タイトル？説明文？カテゴリ？写真処理？）
3. その工程を Claude Code で支援できないか相談
   - 例：「この商品スプレッドシートからeBay出品用のタイトル案と説明文を生成して」
   - 例：「過去の売れ筋商品のタイトル傾向を分析して、新規商品のタイトル案を3つ提案して」

---

## つまづきポイント（社長の体験から）

- **完全自動化より「下書き生成」のほうが現実的**（最終チェックは人間）
- **eBay の出品APIは複雑**（API直叩きは難易度高・最初はCSV一括出品から）
- **画像処理は別ツール**（Claude Code は画像生成より文章生成が得意）
