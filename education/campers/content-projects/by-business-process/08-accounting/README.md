# 08. 経理・外注報酬 — 会計業務のAI化

> このフォルダは「経理・外注報酬・確定申告関連」業務のAI化事例を集める棚です。

---

## この業務段階で何ができるか（メンバー向け）

### 課題（メンバーが抱えがちな悩み）
- 外注スタッフへの月次請求書作成が手動で時間取られる
- 領収書・経費の整理が後回しになる
- 確定申告前に1年分の経費まとめが大変
- 会計ソフト（freee／弥生／マネフォ）への入力が面倒
- 売上データを会計ソフトに転記するのが手作業

### Claude Code でできること
- **月次請求書の半自動作成**（外注スタッフ別に勤怠 → 請求書PDF）
- **領収書の自動分類・記録**（OCR → 科目判定 → 月次集計）
- **eBay売上データの会計ソフト連携**（CSV変換・API連携）
- **経費レポート自動生成**（カテゴリ別・月別・前年比）
- **確定申告用集計の年間棚卸し**（売上・経費・粗利益）

## 社長の体験：月次請求書スキル（要点）

### 何をやったか
- eBay外注3名（本間・清水・佐々木）の月次請求書を半自動作成
- 引数 YYYYMM 省略時は前月を自動判定
- スキル化（slash command 風）でいつでも実行可能

### 取り組みのプロセス
- 各スタッフの勤怠データから時間集計
- 請求書テンプレート（Word/PDF）に自動転記
- メンバーごとの単価設定を管理

### つまづいたこと
- スタッフごとの単価・支払条件のバラツキを管理する仕組みが必要
- 月末締めのタイミング調整

### 学んだこと
- **定型作業は最初に自動化すべき**（毎月同じ作業 = 自動化の宝庫）
- **スキル化（slash command）で運用が定着**

### 数値 Before/After
- 手動運用：3名分 × 30〜60分 = 月90〜180分
- 自動化：3名分 × 数秒 = 実質ゼロ
