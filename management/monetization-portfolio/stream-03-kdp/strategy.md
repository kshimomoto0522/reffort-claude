## 仮ニッチ（Week 1で最終確定）

### ミディアムコンテンツ本のコンセプト案

1. **AI Prompt Journal**
   - AIプロンプトを書き込む手帳＋事例ガイド
   - 120ページ、$9.99
2. **Solopreneur's AI Workflow Planner**
   - 90日間のAIワークフロー導入プランナー
   - 180ページ、$14.99
3. **Claude Code for Creators**
   - クリエイター向けClaude Code活用ガイド
   - 150ページ、$12.99
4. **The AI Productivity Stack**
   - 50のAIツールを業務別に分類したリファレンス
   - 200ページ、$16.99

**理由**：ローコンテンツ（塗り絵、日記帳）は2026年には競争激化。**少し骨のあるコンテンツ＋ガイド付き**が長期勝者。

## 原稿生成パイプライン

```
Claude Codeが章単位で生成：
  /stream-03-kdp/manuscripts/book-01/chapter-01.md
  /stream-03-kdp/manuscripts/book-01/chapter-02.md
  ...

社長が章ごとに確認 → OK
  ↓
全章揃ったら：
  - Word/Google Docs形式に結合
  - フォーマット調整（マージン、フォント、見出し）
  - PDF出力
  - KDPアップロード
```
