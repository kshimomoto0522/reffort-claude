# BayChat 機能仕様・料金プラン

> 親: `services/baychat/ai/CLAUDE.md` から「必要時ロード」として参照される詳細ファイル。

## 主要機能一覧

| 機能 | 内容 |
|------|------|
| 翻訳 | バイヤーメッセージを自動翻訳。日本語→各言語に翻訳して送信。DeepL・Google翻訳から選択可能 |
| テンプレート | よく使う返信文を登録・呼び出し。eBayアカウントごとに設定可能 |
| クーポン | eBayで作成したクーポンコードをチャット画面から表示・送信 |
| リピーター管理 | リピーターマーク表示・購入履歴一覧・過去のやり取り（Chat History）確認 |
| タスク管理 | バイヤー/オーダーごとにチームでタスク共有。担当者・期限設定可能 |
| 商品編集 | 価格・数量・Best Offer・出品取消をチャット画面から直接編集 |
| 自動メッセージ | 売れた時・発送時・配達時・フィードバック時・営業時間外など（詳細は下記） |
| AI Reply | **開発中**（テスト環境のみ。本番リリース準備中） |

## 自動メッセージ機能（詳細）

### 送信タイミング一覧

| タイミング | 内容 |
|----------|------|
| 商品が売れた時 | すぐ〜24時間後まで指定可能。リピーター専用メッセージも設定可能 |
| オファーが成立/拒否した時 | オファー結果に応じて自動送信 |
| 商品を発送した時 | Mark as shippedが対象 |
| 商品が配達された時 | eBay配送状況を反映 |
| 配達完了のN日後 | 1〜90日後まで設定可能 |
| フィードバックをもらった時 | ポジティブ・ニュートラル・ネガティブ別々に設定可能 |
| 営業時間外 | 曜日ごとに設定。定休日はその曜日の24時間。タイムゾーン設定対応 |

### 共通設定・その他

- トラブル（未着・リターン・ケース・Dispute）が発生したオーダーへの自動送信を停止できる
- 対象外バイヤーの個別設定が可能
- インサート機能：バイヤーID・氏名・オーダーID・商品名・追跡番号・配送先住所等を自動挿入
